# OpenMarket

We use HTTParty to send SMS messages using the OpenMarket API.  See USAGE below for details.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'openmarket'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install openmarket
